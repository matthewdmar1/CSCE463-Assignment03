Paint Program
Overview
This project is a simple paint program created using OpenGL and GLFW. The application allows users to draw on a window using a brush of adjustable size and color. Users can also reset the drawing area to a background color of their choice.

Features
Drawing: Click and drag to draw squares with the current brush color and size.
Brush Size: Adjust the brush size using the + and - keys.
Brush Color: Change the brush color by pressing keys 0 to 7, where each number corresponds to a specific color.
Background Color: Change the background color by holding Shift and pressing specific keys:
! - Red
@ - Green
# - Blue
$ - Yellow
% - Cyan
^ - Magenta
& - Gray
* - Black
( - White
Reset: Reset the drawing area to the current background color by pressing the right mouse button.
